## 프로젝트 소개

### 데이팅 매칭 웹사이트 구현

<p align="justify"> 
  - 이상형 매칭 시스템 : 다양한 질문을 통해 나의 이상적인 상대방을 매칭시켜줌<br>
  - 1:1채팅 기능 : 1:1로 원하는 상대방과 실시간 채팅 기능을 지원<br>
  - 회원 랭킹 시스템 : 사이트 회원들이 직접 매기는 회원 평점 시스템

<br>
